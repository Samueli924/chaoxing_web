## :books: 使用方法

### :one: 使用Python环境运行源文件

1. git clone https://github.com/Samueli924/chaoxing_web.git

2. cd chaoxing_web

3. pip install -r requirements.txt

4. python web_listener.py

5. 浏览器访问http://127.0.0.1:7777

6. [使用文档](https://blog.samuelchen.cn/archives/chaoxingwebtutorial)
